ascii-graph

GitHub

A Go library that takes graphs written as strings like:

1   2
 \ /
  5
 / \
3   4

and converts them to Go code like:

g.AddEdge("1", "5")
g.AddEdge("2", "5")
g.AddEdge("3", "5")
g.AddEdge("4", "5")

Originally motivated by confusing unit test code for graph algorithms.
