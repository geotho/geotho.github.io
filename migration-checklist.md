# Blog Migration Checklist

This document tracks the migration of blog posts from `geotho.github.io` to `geotho.io`, following the agreed workflow.

## Process
1. Select the next pending post from this checklist.
2. Export the post content, assets, and metadata into the `geotho.io` structure.
3. Share the migrated post for review and wait for sign-off.
4. Update the checklist (status, notes, links) once the post is approved.
5. Repeat until every post is migrated.
